# 英単語フラッシュカード | English Vocabulary Flashcards

インタラクティブな英単語フラッシュカード学習Webアプリケーション。  
500〜750語の実用的な英単語を3Dフリップアニメーションで楽しく学べます。

## 🌐 デモ

GitHub Pagesで公開後、以下のURLでアクセスできます：  
`https://<username>.github.io/studyenglish/`

## ✨ 機能

- 📚 **750語以上**の実用的な英単語を収録
- 🎯 **3段階の難易度**（初級・中級・上級）
- 🔢 **学習単語数の選択**（10 / 25 / 50 / 100 / 250 / 500語）
- 🔀 **順番通り or ランダム**出題
- 🃏 **3Dフリップアニメーション**でカードをめくる
- ⌨️ **キーボードショートカット**（←→キーで移動、スペースでフリップ）
- 📱 **スワイプ操作**対応（スマートフォン）
- 📊 **進捗バー**で学習状況を可視化
- 🎉 **完了画面**で学習結果を表示
- 📐 **レスポンシブデザイン**（PC・タブレット・スマートフォン対応）

## 🚀 使い方

1. 学習する**単語数**を選択
2. **出題順序**を選択（順番通り or ランダム）
3. **難易度レベル**を選択
4. 「**学習をはじめる**」ボタンをクリック
5. カードをクリック/タップして**意味と例文**を確認
6. 前へ/次へボタンで移動

## 🛠️ 技術スタック

- HTML5
- CSS3（3Dトランスフォーム、カスタムプロパティ）
- Vanilla JavaScript（フレームワーク不要）
- Google Fonts（Inter, Noto Sans JP）
